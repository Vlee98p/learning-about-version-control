# Learning about Version Control
Author: Vienne Lee <br>
Solving merge conflicts takes conscious effort, but I can do it! <br>

What you like most about version control is that I can work on a project without worrying about losing my work while collaborating with others, and I can keep track of all the changes. <br>
The most challenging/confusing/frustrating about version control is that every commit message needs to be clear and concise. Sometimes the changes are so small that it is difficult to come up with a meaningful commit message.
